# 🔍 Search Algorithm Comparison

Perbandingan performa **Binary Search** dan **Linear Search Rekursif** menggunakan Python dengan visualisasi grafik waktu eksekusi.

---

## 📖 Deskripsi

Project ini dibuat untuk menganalisis efisiensi algoritma pencarian dengan membandingkan:

- **Binary Search (Iteratif)**
- **Linear Search (Rekursif)**

Program menggunakan data dummy berupa kumpulan nama unik, kemudian mengukur waktu eksekusi masing-masing algoritma dan menampilkan hasilnya dalam bentuk grafik.

---

## 🚀 Features

- Binary Search implementation
- Recursive Linear Search implementation
- Execution time benchmarking
- Performance visualization using Matplotlib
- Comparison across multiple input sizes

---

## 🧠 Algorithm Complexity

| Algorithm | Best Case | Average Case | Worst Case |
|---|---|---|---|
| Binary Search | O(1) | O(log n) | O(log n) |
| Linear Search | O(1) | O(n) | O(n) |
